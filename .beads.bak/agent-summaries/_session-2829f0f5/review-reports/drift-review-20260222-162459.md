### DRIFT-001: `reviews.md` reader attribution — stale "Pantry (review mode)" in 4 locations

**File:Line**: `orchestration/RULES.md`:47
**Severity**: P2
**Category**: Stale cross-file reference

**Description**: RULES.md:47 (in the FORBIDDEN reads list) says `reviews.md` is "read by Pantry in review mode". The Pantry review mode (`pantry-review`) was deprecated and replaced by `build-review-prompts.sh`. The Pantry no longer reads `reviews.md`; `build-review-prompts.sh` reads it instead.

The same stale attribution appears in:
- `orchestration/RULES.md`:440 (Template Lookup table: "Review details (read by the Pantry)")
- `README.md`:352 (File reference table: "The Pantry (review mode)")
- `README.md`:252 (Information diet section: "`reviews.md` [is] read by the Pantry")
- `orchestration/GLOSSARY.md`:82 (Pantry role description: "Reads implementation or review templates")

**Impact**: An operator reading the FORBIDDEN list in RULES.md would understand the Pantry still handles review prompt composition and might attempt to spawn `pantry-review` for Step 3b. The FORBIDDEN list is a "Queen never reads" list — it's listing what agents read; having an incorrect reader attribution there is drift, not merely cosmetic.

**Suggested fix**:
- `RULES.md`:47 — change to: `- \`orchestration/templates/reviews.md\` — Review protocol (read by \`build-review-prompts.sh\` at Step 3b-ii)`
- `RULES.md`:440 — change "Review details (read by the Pantry)" to "Review details (read by `build-review-prompts.sh` at Step 3b)"
- `README.md`:352 — change "The Pantry (review mode)" to "`build-review-prompts.sh` (Step 3b-ii)"
- `README.md`:252 — change "`implementation.md` and `reviews.md` are read by the Pantry" to "`implementation.md` is read by the Pantry; `reviews.md` is read by `build-review-prompts.sh` at Step 3b"
- `GLOSSARY.md`:82 — change "Reads implementation or review templates" to "Reads implementation templates"

### DRIFT-002: README Hard Gates table missing SSV checkpoint

**Category**: Stale cross-file reference / incomplete propagation

**Description**: The README Hard Gates table lists 4 checkpoints: CCO, WWD, DMVDC, CCB. ...
