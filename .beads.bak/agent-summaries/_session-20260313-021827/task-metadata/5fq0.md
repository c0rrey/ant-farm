## Root Cause
Architect workflow switches to embedded Dolt mode, runs dep add commands, then switches back to server mode. If dep add fails mid-sequence, system is left in embedded mode, causing subsequent bd calls to fail.

## Expected Behavior
Server mode restoration should run unconditionally even on dep add failure.

## Acceptance Criteria
1. `bd dolt set mode server && bd dolt start` runs even if `bd dep add` commands fail
2. Failure to enter embedded mode produces a clear error and does not proceed with dep commands
3. A warning is emitted if server mode restoration fails
