Tradeoffs: Higher blast radius, harder to audit, no benefits over targeted edits.

Use patterns matching "data file" only when preceded by specific context words (e.g., "Pantry", "consolidation"). Would require grep to confirm patterns first, then targeted edits anyway. Adds complexity with no benefit over Approach A given the small number of occurrences.

Tradeoffs: Overly complex for a small, well-understood set of changes. Approach A achieves the same precision with less overhead.
