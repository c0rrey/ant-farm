## Root Cause
docs/installation-guide.md:193 uses 'rm ~/.git/hooks/pre-push' which resolves to the user's home directory .git, not the repo's .git. Any user running this verbatim will either hit 'No such file' or delete a hook from the wrong repo.

## Expected Behavior
Uninstall path is .git/hooks/pre-push (relative to repo root). Both hooks have uninstall instructions.

## Acceptance Criteria
1. Uninstall path is .git/hooks/pre-push
2. Both hooks (pre-push and pre-commit) have uninstall instructions
